    Olá! seja bem vindo ao meu projeto de Lista de tarefas

    Desenvolvi esse projeto para estudos sobre programação web ulizando a linguagem JavaScript. Para nós programadores sempre que iniciamos nossos estudos o primeiro passo é fazermos o famoso CRUD (Create, Read, Update, Delete), criar, ler, atualizar e deletar, que são as quatro operações básicas do desenvolvimento de uma aplicação.

    Esse projeto não tem conexão com um banco de dados pois eu queria que ele ficasse disponível para qualquer usuário sem a necessidade de instalação e obviamente de forma gratuita.

    Optei por utlizar o localStorage para manter os dados salvos mesmo quanto o usuário saia do navegador.

    Se você é um usuário que apenas quer utilizar a lista de tarefas fique a vontade ela é 100% funcional e gratuita.

    Se você é um programador ou está estudando e ficou interessado no projeto pode ter acesso ao meu código fonte disponível no meu repositório público no GitHub.
